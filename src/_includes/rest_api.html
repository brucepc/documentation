REST APIs ¶

The REST APIs provide programmatic access to read and manage SumUp accounts details, transaction reporting and checkouts. The API identifies applications and users using Oauth.

APIs access ¶

In order to use SumUp API’s you need to regiser and setup your application credentials. Valid requests require Bearer access token supplied in the standard http authorization header.

Overview ¶

API endpoints and operations are described in the following documents:

  • Accounts API - Allows you to manage merchant accounts and their assets

  • Transactions API - Extended transaction history and transaction details

  • Checkouts API - Create and process checkouts. See also Checkout flow
